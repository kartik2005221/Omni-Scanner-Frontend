Project Overview

OmniScanner is a powerful, Python-based network scanning suite crafted for ethical hackers, sysadmins, and cybersecurity learners. It combines both CLI and Web interfaces for seamless scanning, analysis, and reporting of networks.
With features like ARP scans, ping sweeps, traceroutes, and advanced Nmap integrations — OmniScanner empowers users to discover hidden devices, map routes, identify services, and uncover potential vulnerabilities with ease.
Whether you’re securing your home network or exploring enterprise setups, OmniScanner is your lightweight but mighty companion for ethical network reconnaissance.

Our Vision

OmniScanner’s core objective is to simplify and democratize network scanning — putting powerful tools in the hands of students, cybersecurity enthusiasts, and professionals alike. We aim to bridge the gap between raw technical complexity and beginner-friendly usability, while still maintaining full control and transparency for advanced users.

Tech Used

Core Technologies

Omni-Scanner is powered by Python, leveraging modules like subprocess, os, and argparse to execute and manage low-level network tasks with precision. OS-specific scripts are separated cleanly to ensure consistent behavior on Windows and Linux platforms.

Web Interface & CLI

The frontend is built using HTML5, Tailwind CSS, and Jinja templating via Flask, ensuring a clean, responsive UI. Each scan result is styled dynamically with chips, color-coded highlights, and modern components for an intuitive scanning experience.

Backend & Logic Layer

Flask serves as the backend framework, binding user interactions to core scan logic. Routes are structured modularly to handle tasks like scanning, history logging, and error feedback. Python handles data parsing and shell interaction for every scan type.
