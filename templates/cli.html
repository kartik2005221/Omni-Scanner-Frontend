For Linux users, you might need to use sudo for full network access.

Common Tips & Tricks

  • Ctrl + C to gracefully exit the tool anytime.
  • Make sure Nmap is installed and added to your PATH (especially on Windows).
  • If using on Linux, install arp-scan via package manager for local device discovery.
  • Admin access is recommended for advanced scans like ARP, traceroute, and OS detection.

Trouble? Try These

  • If ModuleNotFoundError, check Python version and virtual environment.
  • If PermissionError, re-run with sudo or admin mode.
  • If command not found, verify dependencies like nmap or arp-scan are installed.
  • Windows users: Make sure Npcap is installed and service is running.
